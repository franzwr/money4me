    Destroys a bill. This actions can't be performed if the bill has a payment associated.

    # File app/controllers/api/cuenta_controller.rb, line 62
def destroy
        @cuenta = Cuenta.find_by(id_cuenta:  params[:id],
                     id_empresa: current_empresa.id)
        if @cuenta.pago && @cuenta.destroy
                render :json => {}
        else
                render :json => {}, status: :internal_server_error
        end
end
    index() click to toggle source

    Main action. Shows all bills, with all meaningfull relationships. If a folio is provided, acts as the show action with the folio number as filter.

    # File app/controllers/api/cuenta_controller.rb, line 20
def index
if params[:id_propio_empresa]
                @cuenta = Cuenta.find_by :id_propio_empresa => params[:id_propio_empresa]
        else
                @cuenta = Cuenta.all
        end
        if @cuenta
                render :json => @cuenta, :include => [:empresa, :pago]
        else
                render :json => {}, status: :not_found
        end
end
