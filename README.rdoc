* Configuration

Por defecto, la base de datos local debe estar configurada en el puerto 3306,
con user: root y password: local. El archivo schema.rb genera solo las tablas 
correspondientes a esta base de datos, por lo que no intenta modificar la base
de datos remota. Los valores por defecto pueden cambiarse en config/database.yml. 

Base de Datos

Para crear e inicializar la base de datos se utiliza rake db:setup.
